This application can be used as a means of 'predicting' the effects of different funding in the educational system of a country
on its illiteracy rates*.
The illiteracy rate is represented by the percentage of illiterate people in a specific country.

Input: The name of the country in which a change of funding is to be tested
and the amount of money to be invested (expressed as % of GDP).
Output: The rate of illiteracy given the change in funding.

**In young adults and older (15+ year olds)



Created using Python and its libraries to implement the Linear Regression Algorithm.

Description

Project built by Antonio Berbece (2020) as a means of understanding how Linear Regression works.
In addition, this also emphasizes framework usage (Eel, Jquery).

Algorithm Description:
Multiple Linear Regression which uses (for now) the % GDP allocated for education (in total)
and the gross primary school enrollment percentage.

The algorithm predicts the change in the rate of illiteracy, the score of the current model being 80%..
